## Sunrise Weather App

**Sunrise** is a modern Android weather application that provides real-time weather information using the **OpenWeatherMap** service.

🔗 **OpenWeatherMap Website:**  
https://openweathermap.org/

---

## 🔑 API Key Setup

To use the app, you must add your own OpenWeatherMap API key.

1. Go to https://openweathermap.org/
2. Create a free account
3. Generate an API key
4. Add the API key to the project  
   (in 'utils.Constants' file)

⚠️ **The app will not work without a valid OpenWeatherMap API key.**
